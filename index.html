Sobre mim

Meu nome é Franklin Ferreira, sou pernambucano e impulsionado por aprender coisas novas, ensinar e resolver problemas com tecnologia. Minha formação se iniciou na licenciatura em Química (UFRPE) e tive a oportunidade de fazer intercâmbio para University of Manchester por um ano, onde consegui trabalhar com o estado da arte da minha área, conhecer outras culturas e aprimorar meu inglês. Me estabeleci na área de Física computacional, fazendo mestrado na UFRPE e sou atualmente doutorando em Física na UFPE. Esse percurso me levou a conhecer a área de dados e seus desafios. Atualmente, trabalho como cientista de dados na área de crédito na Neurotech.

Projetos

Classificação de diabetes

Projeto end-to-end de classificação utilizando machine learning com o objetivo prever o diagnóstico de diabetes a partir um conjunto de informações de saúde de um paciente. O projeto inclui um modelo heurístico de baseline, uso de pipelines, validação cruzada estratificada e feature selection. Além disso, também foi realizado um estudo do impacto de negócio do modelo e deploy no HugginFace.

Regressão de renda

Projeto end-to-end de previsão de renda de um grupo familiar baseado em dados do Censo Brasileiro de 2000. Utilizei SQL para extração de dados (BigQuery), fiz etapas de pré-processamento, EDA, tratamento de outliers, feature engineering. Finalmente, foram desenvolvidos modelos de ML (Random Forest, XGBoost, LGBM) e optimização de hiperparâmetros com bayesian optimization.

Clusterização de clientes

Projeto de segmentação de clientes utilizando algorítimos de clusterização. Uma aplicação desse tipo de modelo é obter características de grupos de clientes de modo a auxiliar o planejamento de estratégias de negócio e marketing direcionados. Usa KMeans e KPrototypes para clusterização e PCA para redução de dimensionalidade.

Classificador de Spam

Classificador de emails como legítimos (ham) ou falsos (spam) com ferramentas de processamento de linguagem natural (NLP). O projeto foi dividido em etapas de ETL e modelagem. Na ETL, os dados foram obtidos via requisição, transformados em emails e categorias, e salvos em csv. Na modelagem os dados foram pré-processados, analisados (análise de sentimentos, wordcloud, etc) e modelados com machine learning e deep learning.

Modelos de Machine Learning

Projeto de reescrita de algoritmos de machine learning para entendimento teórico. Entre os modelos descritos estão KNN, regressão linear, árvores de decisão, K-Means e Logistic Regression, incluindo vídeos de explicação dos modelos.

Recomendações de filmes

Os sistemas de recomendação são utilizados para filtrar entre um catálogo de possibilidades, decidindo quais devem ser apresentadas ao usuário com base em informações do catálogo ou dos usuários. Esses sistemas podem ser usados para aprimorar a experiência do usuário, tornando o conteúdo disponível relevante para o usuário. Neste projeto, vamos explorar um conjunto de dados de filmes e apresentar alguns dos principais tipos de sistemas de recomendação.
